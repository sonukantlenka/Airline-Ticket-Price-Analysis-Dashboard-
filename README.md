#  Airline Ticket Price Analysis Dashboard

A data-driven dashboard that analyzes airline ticket prices based on multiple factors such as airline, class, stops, and booking time.

##  Overview

This project presents an interactive **Airline Ticket Price Analysis Dashboard** that helps users understand pricing trends across different airlines and travel conditions.
The dashboard provides insights into:
* Ticket price variations
* Revenue generation
* Flight distribution
* Impact of booking time and stops on pricing

##  Key Features

*  **Total Flights Overview**
  * Displays total number of flights analyzed
* 💰 **Revenue Insights**
  * Shows total revenue generated from ticket sales
* 🧾 **Average Ticket Price**
  * Separate analysis for:
    * Business Class
    * Economy Class
* 🛫 **Airline Comparison**
  * Compare average prices across airlines like:
    * Air India
    * Indigo
    * Vistara
    * SpiceJet
    * GO FIRST
    * AirAsia
* ⏱ **Departure Time Analysis**
  * Price trends based on:
    * Morning
    * Afternoon
    * Evening
    * Night

*  **Stops Analysis**
  * Impact of:
    * Non-stop
    * One stop
    * Two or more stops

*  **Price vs Days Left**
  * Understand how ticket prices change as departure date approaches
* 🧮 **Class Distribution**
  * Number of flights by class (Business vs Economy)

##  Tech Stack
*  **Power BI** – Dashboard creation & visualization
*  **Data Source** – Airline ticket dataset (CSV/Excel)
*  **Data Processing** – Data cleaning & transformation


##  Insights You Can Derive
* Which airline has the highest average ticket price
* How ticket prices vary between business and economy class
* Best time to book flights for lower prices
* Impact of stops on ticket pricing
* Revenue contribution by different classes
